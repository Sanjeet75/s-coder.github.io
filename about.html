The heart of Coder

Code-server is the primary open source project we maintain. It allows developers to use a browser to access remote dev environments running VS Code. Coder builds upon the success of code-server and adds features designed for enterprise teams including support for additional IDEs and advanced security features. Coder is dedicated to keeping developers in flow — removing the obstacles that prevent them from writing code at the speed of thought.
